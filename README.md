# 应急转贷业务管理系统静态原型

这是从 `yjzd.zip` 恢复出的前端静态原型项目。项目目前不依赖后端服务，页面数据、审批记录、客户信息和报表内容均写在本地 HTML 中，JavaScript 主要负责页签切换、详情页展示、弹窗、分页等交互。

## 项目结构

```text
yjzd/
├─ index.html            # 页面主体和静态业务数据
├─ css/
│  └─ style.css          # 页面样式
├─ js/
│  └─ app.js             # 前端交互逻辑
├─ images/
│  ├─ logo.png           # 系统 logo 占位图
│  └─ avatar.png         # 用户头像占位图
└─ docs/
   └─ 项目文档.md        # 项目说明文档
```

## 运行方式

方式一：直接双击 `index.html` 打开。

方式二：使用本地静态服务器预览：

```powershell
cd /d D:\code\yjzd
python -m http.server 8080
```

然后在浏览器打开：

```text
http://localhost:8080
```

## 当前功能

- 首页仪表盘：消息中心、运行指标、累计服务成效。
- 业务全流程：展示应急转贷项目的授信、合同放款流程。
- 业务审批：授信审批、业务变更处理、审批详情、同意/退回/拒绝弹窗。
- 放款审批：放款申请列表、放款详情、放款记录、撤回确认。
- 报表上报审批：月度经营指标报表查看、审批和历史记录。

## 注意事项

- 当前项目是静态原型，不包含登录、权限、接口请求、数据库或真实审批流。
- 页面内存在示例企业、手机号、身份证号等字段，正式对外演示前应确认是否需要脱敏。
- 如果后续要开发成真实系统，建议先拆分页面组件、定义接口文档，再接入后端业务服务。
